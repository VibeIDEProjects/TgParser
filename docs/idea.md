# TgParser — Идея проекта

<!-- vibeVersion: 1.0.0 -->

## Суть

Написать парсер Telegram-каналов — утилиту для извлечения сообщений.

## Что парсим

| Тип канала | Источник данных | Особенность |
|------------|-----------------|-------------|
| Открытые | MTProto API (Telethon) | Прямой доступ к сообщениям |
| Закрытые | Web-версия Telegram | HTML-вёрстка, проще разобрать |

## Проблемы закрытых каналов

- Часто включён **запрет копирования** (copy protection)
- Нужен обход: снятие CSS-ограничений, отключение JS-обработчиков, прямое чтение DOM

## Авторизация

- При входе в web-версию Telegram генерируется **QR-код**
- Пользователь сканирует QR на телефоне → авторизация подтверждается
- Сессию нужно сохранять для переиспользования

## Примечания

- Парсинг через web предпочтителен для закрытых каналов — HTML отдаётся готовым, не нужно разбирать MTProto-схемы
- Нужно предусмотреть сценарий протухания сессии (реавторизация)
