# scripts/

Диагностические/отладочные скрипты, которые **запускаются вручную** —
не часть `pytest` и не runtime-код приложения.

> **Если вы перенаправляете вывод в файл — пишите его в `../logs/`, а не сюда.**

Правильно:
```bash
python scripts/dump_defs.py > ../logs/dump_defs.txt 2>&1
```

Неправильно:
```bash
python scripts/dump_defs.py > dump_defs.txt 2>&1    # свалка в scripts/
```

## Где что лежит

| Папка          | Назначение                                          |
| -------------- | --------------------------------------------------- |
| `scripts/`     | `.py` отладочные скрипты (только они)               |
| `logs/`        | `.log`/`.txt` вывод приложения и скриптов           |
| `tests/`       | `test_*.py` автотесты (pytest)                      |
| `data/output/` | Результаты парсинга (`.json`, `.csv`, `.md`)        |
